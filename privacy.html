4. Data Storage & Security

4.1 Where We Store Your Data

  • Firebase Firestore: Package tracking data, user preferences, subscription status
  • Chrome Local Storage: Extension settings and cached data
  • Secure backend server: Temporary processing of email analysis

4.2 Security Measures

  • All data transmission uses HTTPS/TLS encryption
  • Firebase Authentication for secure login
  • API tokens are securely stored and never exposed
  • Regular security audits and updates

4.3 Data Retention

  • Package data: Kept for 30 days after delivery (or longer if you save it)
  • Account data: Retained while your account is active
  • Temporary email analysis: Deleted immediately after processing
  • Logs: Kept for 90 days for debugging purposes
